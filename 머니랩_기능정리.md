# 머니랩(MoneyLab) 웹사이트 기능 정리

---

## 공통 요소 (전 페이지)

### 네비게이션 바 (Navbar)
- CSS Grid 3컬럼 레이아웃 (로고 | 메뉴 | 우측 버튼)
- 로고: "GGU" (G에 오렌지 컬러 #FF4D00)
- 메뉴 링크: 구독하기, 후기, 자주 묻는 질문
- "구독하기" 옆 오렌지 점(dot) 표시
- 로그인 상태별 우측 영역 분기:
  - 비로그인: "지금 시작하기" CTA 버튼
  - 로그인: "내 구독정보" 링크 + "로그아웃" 버튼
- 모바일(768px 이하): 햄버거 메뉴 → 슬라이드인 패널

### 푸터 (Footer)
- 회사 정보: 대표, 주소, 사업자등록번호, 통신판매업신고번호
- 연락처: 이메일, 전화번호
- 링크: 이용약관(모달), **개인정보처리방침**(모달, 굵기 적용)
- SNS 아이콘: 인스타그램, 쓰레드, 유튜브, 블로그
- 면책 고지: 투자 손익 귀책 안내, 단방향 서비스 고지

### 법적 문서 모달
- 이용약관 모달: 서비스 목적, 정의, 효력, 가입, 해지, 요금, 환불 등
- 개인정보처리방침 모달: 수집 목적, 항목, 보유기간, 제3자 제공, 파기, 권리 등

### 로그인 상태 관리
- localStorage 키: `moneylab_logged_in` (true/false)
- 무료체험 상태: `moneylab_trial_used` (true/false)
- Sim Toggle: 화면 우측하단 고정 버튼으로 로그인/로그아웃 상태 테스트

## 기술 스택

### 현재 프로토타입 (HTML)
- 프레임워크 없음 (순수 HTML/CSS/JS, 단일 파일 구조)

- 반응형: 768px / 900px 브레이크포인트

- Next.js 16 App Router
- React 19 + TypeScript 5
- Tailwind CSS 4
- 폰트: Pretendard (next/font 또는 CDN)
- 이미지: public/ 디렉토리 기준, next/image 사용
- 내부 라우팅: next/link (/, /payment, /review, /faq, /login, /profile, /terms)

## 홈페이지 TSX 컴포넌트 구조 (HomePage.tsx)

### 파일 구성
- `HomePage.tsx` — 홈 본문 전체 ("use client" 컴포넌트)
- `tailwind-extend.css` — fade-up 애니메이션 및 딜레이 유틸리티 (@import 필요)

### 컴포넌트 트리
```
<HomePage>            ← page.tsx에서 default export
  <HeroSection />     ← 히어로 (그라데이션 + CTA + 스크롤 힌트)
  <PortfolioSection/> ← 포트폴리오 무한 슬라이더
  <ServiceSection />  ← 서비스 소개 3카드 그리드
  <CeoSection />      ← 대표 소개 (다크 배경)
  <ReviewsSection />  ← 후기 슬라이더 + 모달
  <PartnersSection /> ← 파트너 로고
  <CommunitySection/> ← 커뮤니티 사진 그리드
  <AboutSection />    ← 통계 카운트업 (IntersectionObserver)
  <CtaSection />      ← 최하단 CTA (다크 배경)
</HomePage>
```

### 커스텀 훅
- `useInfiniteSlider(itemCount, autoMs)` — 포트폴리오/후기 무한 슬라이더 로직 (자동재생, 일시정지)

## 구독하기 TSX 컴포넌트 구조 (PaymentPage.tsx)

### 컴포넌트 트리
```
<PaymentPage>           ← /payment 라우트 page.tsx
  <PlanHero />          ← 로고 + 카테고리 탭 (주식 / 주식+코인)
  <PricingCards />      ← 2컬럼 플랜 카드 (베이직 / 프리미엄)
  <BillingToggle />     ← 연/월 결제 토글
  <ShowcaseSection />   ← 혜택 슬라이더 (다크, 5개 슬라이드)
  <FaqSection />        ← FAQ 아코디언 + 우측 이미지
  <CheckoutModal />     ← 결제 확인 팝업 (자동결제 10% 할인)
  <FeatureTooltip />    ← 기능 클릭 시 상세 툴팁
</PaymentPage>
```

### 주요 상태 (useState)
- `currentCategory` — '주식' | '주식 + 코인'
- `isYearly` — 연/월 결제 토글
- `checkoutOpen` + `checkoutData` — 결제 모달
- `faqOpenIndex` — FAQ 아코디언
- `tooltipData` — 기능 툴팁

### 가격 계산 로직
- 월 결제: `monthly` 가격 그대로, 하단 "하루 약 X원" = `Math.floor(monthly / 30)`
- 연 결제: `yearlyMonthly` 가격 표시, 절약액 = `(monthly - yearlyMonthly) * 12`
- 체크아웃 총액: 연간 `yearlyMonthly * 12`, 월간 `monthly`
- 자동결제 할인: 총액의 10%

- 요약: 블루 배경, 3개 포인트
- 본문: 2줄 clamp + "더보기" 토글
