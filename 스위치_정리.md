### 2. 구독 상태 (3단계 사이클)

| 상태 | 스위치 텍스트 | 클릭 시 |
|------|-------------|--------|
| 체험중 | ✅ 체험중(현재) | → 구독만료 |
| 구독만료 | 🚫 구독만료(현재) | → 미구독자 |
| 미구독자 | ⬜ 미구독자(현재) | → 체험중 |

**체험중(현재) 상태**
- 구독카드: "주식+코인 (프리미엄)" + 파란 뱃지 "무료 체험중(14일)"
- 구독기간: 2025.08.01 ~ 2027.02.28
- 채널 입장하기 버튼: 보임
- 이용 가능한 혜택: 4개
- 혜택 이용하기 버튼: 활성화

**구독만료(현재) 상태**
- 구독카드: "주식+코인 (프리미엄)" + 빨간 뱃지 "구독만료"
- 채널 입장하기 버튼: 숨김
- 이용 가능한 혜택: 없음
- 혜택 이용하기 버튼: 비활성화 → "플랜 업그레이드시 이용가능"
- 체험하기 버튼: 비활성화 → "플랜 업그레이드시 이용가능"

**미구독자(현재) 상태**
- 구독카드: "미구독자" (뱃지 숨김)
- 구독기간: 없음
- 채널 입장하기 버튼: 숨김
- 무료로 채널 입장 버튼 클릭 시: "지금 무료체험을 시작하시겠습니까?" 팝업 표시

### 3. TG 입장 상태

| 상태 | 스위치 텍스트 |
|------|-------------|
| 미입장 | ⬜ TG미입장(현재) |
| 입장완료 | ✈️ TG입장완료(현재) |

**TG미입장 상태**
- 온보딩 히어로 섹션 표시 (채널 입장 안내)
- 입장 완료 액션 영역 숨김

**TG입장완료 상태**
- 온보딩 히어로 섹션 숨김
- "채널 입장하기" + "구독 연장하기" 버튼 표시

---

### 4. 베이직/프리미엄 혜택 플로우

| 상태 | 스위치 텍스트 |
|------|-------------|
| 베이직 | ⬜ 베이직 혜택 플로우(현재) |
| 프리미엄 | 👑 프리미엄 혜택 플로우(현재) |

**베이직 혜택 플로우 상태**
- 이용 가능한 혜택: 4개
- 잠금 혜택 섹션: "🔒 체험만 가능한 혜택 (플랜 업그레이드시 이용가능)" 표시
- 잠금 카드에 프리미엄 잠금 뱃지 표시
- 체험하기 버튼 표시
- 분석 제안 버튼: "플랜 업그레이드시 이용가능" (비활성화)

**프리미엄 혜택 플로우 상태**
- 이용 가능한 혜택: 8개
- 잠금 섹션 및 업그레이드 안내 숨김
- 모든 혜택 카드 활성화 (잠금 뱃지 제거)
- 체험하기 버튼 → "혜택 이용하기 →"
- 분석 제안 버튼: "제안하기" (활성화)
